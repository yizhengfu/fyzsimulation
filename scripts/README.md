# scripts — 后处理/分析脚本

ReaxFF 反应力场后处理、ReacNetGenerator 工具和机器学习辅助脚本。

## 目录结构

```
scripts/
├── Python/                   # Python 脚本合集
│   ├── 反应力场处理/          # ReaxFF 结果分析（9个脚本）
│   ├── LJ.py                 # Lennard-Jones 势能计算
│   ├── hello.py              # 测试
│   ├── pair.txt              # 配对参数
│   └── python / python2      # Python 环境路径记录
├── python1/
│   └── 11.py                 # 单脚本
├── reacnetgenerator/         # ReacNetGenerator 工具
│   ├── 反应网络图形化界面.py # 反应网络可视化
│   ├── 预处理图形化界面.py   # 轨迹预处理
│   ├── 预处理图形化界面bond.py # 基于成键的预处理
│   ├── 势能处理.py           # 势能曲线处理
│   ├── 自制反应网络.py       # 自定义反应网络生成
│   └── 视频讲解.mp4          # 使用教程
├── 分子式.py                 # RDKit 分子式/结构绘图
├── ZDB.py                    # PDF 批量转 Word（PyMuPDF）
└── 机器学习/
    ├── lcurve.out             # DeepMD 训练曲线数据
    └── try                    # 测试脚本
```

## 反应力场处理脚本

| 脚本 | 用途 |
|------|------|
| `主要产物.py` | 统计 ReaxFF 产物分布 |
| `反应时间.py` | 反应产物的时间演化分析 |
| `反应路径_指定目录.py` | 指定目录的反应路径提取 |
| `物种数分子数.py` | 物种数和分子数统计 |
| `团簇_时间.py` | 最大团簇尺寸的时间演化 |
| `标记最大团簇2.0.py` | 标记并追踪最大团簇 |
| `删除可逆反应.py` | 过滤可逆反应（保留净反应） |
| `箭头替换.py` | 反应式箭头格式转换 |
| `精准分列.py` | ReaxFF 输出文件的精确分列 |
| `计算C个数.py` | 统计碳原子数变化 |
