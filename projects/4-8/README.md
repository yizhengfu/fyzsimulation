## 子目录详情

### MOF 膜层间距系列（0v1 ~ 1v9）

**目的**: 研究不同层间距比和层数比对 CH₄/H₂O 扩散行为的影响。

命名规则：`v` 前后数字分别表示施压层和扩散层的气体分子层数。例如：
- `1v1` = 施压层1层 + 扩散层1层
- `0v1` = 施压层0层（只有顶层施压）+ 扩散层1层
- `1v3` = 施压层1层 + 扩散层3层
- `0v1-1` / `1v0-1` = 减半厚度版（更短扩散路径）

公共参数：
- MOF 层厚（vDMofth）≈ 24.6Å（固定）
- 气体类型：CH₄, H₂O
- 分析方法：MSD, RDF, 密度分布, 通量
- 输入模板：Layer.car（MS 建模）→ Layer.data（LAMMPS 格式）
- 运行脚本：in.Diff.txt

### 1v1onelayer（单层膜）

较完整的数据集：298K 和 348K 两个温度，每个温度下 5MPa/10MPa 两压差。
包含 Equ（平衡）和 fin（最终）两个阶段的结果。

### 4-15（混合材料对比）

| 子目录 | 材料类型 | 说明 |
|--------|---------|------|
| `4.68/` | 层间距 4.68Å 的 MOF | 单一间距参数对比 |
| `Graphene/` | 石墨烯膜 | 含 He（氦测漏）对比 |
| `NA/` | NA 型 MOF | 298K 和 323K, 5/10MPa |
| `PCNA/` | PCNA 聚合物复合 | 298K, 含 NPT 对比 |
| `Zif/` | ZIF 材料 | 283K 和 298K, 5/10MPa |

### Graphene（石墨烯膜）

不同层间距的石墨烯膜气体扩散：
- `1-5.6/` — 间距 5.6Å
- `2-4.92/` — 间距 4.92Å
- `3-4.68/` — 间距 4.68Å
- `3.7/` — 间距 3.7Å
- `4.0/` — 间距 4.0Å + 4-13 子系列（含 0v1/1v0/1v1 不同层数比, 298K 和 398K）

每个间距下按层数比分 `ov1/`, `1v0/`, `1v1/`。

### KA 系列（KA-0.5-4.394, KA-0.5-4.895, KA4.394, KA4.895）

**KA 型多孔材料**的气体扩散模拟。
- 编号表示材料参数：`4.394`/`4.895` 指 MOF 层间距，`0.5` 指配比
- KA4.394/KA4.895：348K 温度下有完整仿真结果（含通量 fluxhi/fluxlo 分析）
- KA-0.5-4.895/4-12/：含完整扩散结果（MSD, RDF, 质量密度等）

### PC系列（PCANA, PCKA）

**聚合物复合膜**: MOF 颗粒均匀分散在聚合物基质中。
- PCANA = Polymer Composite + NA 型 MOF
- PCKA = Polymer Composite + KA 型 MOF

### Zif

**ZIF 多孔材料**的 CH₄/H₂O 扩散。

## 输入输出文件说明

每个模拟目录包含：

| 文件 | 类型 | 用途 |
|------|------|------|
| `Layer.car` | 输入 | Materials Studio 建模文件（原子坐标+晶胞） |
| `Layer.mdf` | 输入 | MS 力场分配文件 |
| `Layer.data` | 输入 | LAMMPS data 文件（由 MS 转换） |
| `in.Diff.txt` | 输入 | LAMMPS 扩散模拟脚本 |
| `Layer.lammps` | 输入 | LAMMPS 结构文件 |
| `All_*.lammpstrj` | 输出 | 全原子轨迹（平衡/扩散/通量） |
| `CH4_*.lammpstrj` | 输出 | CH4 分子轨迹 |
| `H2O_*.lammpstrj` | 输出 | H₂O 分子轨迹 |
| `*Count_*.txt` | 输出 | 分子数统计 |
| `*MSD*.txt` | 输出 | 均方位移（扩散系数） |
| `*RDF*.txt` | 输出 | 径向分布函数 |
| `*Energy_*.txt` | 输出 | 能量变化 |
| `*Interact_*.txt` | 输出 | 相互作用能 |
| `*massdensity` | 输出 | 质量密度分布 |
| `log.lammps` | 输出 | LAMMPS 运行日志 |
