#Code for Evaporation of LJ Fluid 如
units lj
atom_style sphere
boundary p p f
pair_style lj/cut 2.5 

read_data data.equilibrium
pair_coeff * * 1.0 1.0 2.5
velocity all create 1.0 87287 loop geom
neighbor 1 multi 
neigh_modify delay 0
comm_modify mode multi
region vapor block 0.0 30.0 0.0 30.0 60.0 80.0 units box 
region substrate block 0.0 30.0 0.0 30.0 0.0 1.0 units box 
group bottom region substrate
fix 1 all nve
fix 2 bottom langevin 1.0 1.0 0.1 437624 
fix 3 all evaporate 100 100 vapor 56789
compute cc1 all chunk/atom bin/1d z center 2.0
fix 4 all ave/chunk 100 10 1000 cc1 density/number norm sample file density.profile
fix 5 all ave/chunk 100 10 1000 cc1 temp norm sample file temp.profile
thermo_style custom step temp epair etotal press vol density 
thermo 1000
dump a all atom 1000 dump.evaporate 
dump_modify a sort id 
thermo_modify lost ignore flush yes
run 100000
write_data data.evaporate
